# 📈 Sales Forecasting Dashboard using Machine Learning

This project is an end-to-end Sales Forecasting System built using Python, XGBoost, and Streamlit. The goal of the project is to analyze historical sales data and predict future sales trends using Machine Learning techniques.

## 🚀 Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Lag Features & Rolling Averages
* XGBoost Regression Model
* Future Sales Forecasting
* Interactive Streamlit Dashboard
* Performance Evaluation using MAE, RMSE, and R² Score

## 📊 Model Performance

* MAE improved from **284 → 259**
* RMSE improved from **626 → 580**
